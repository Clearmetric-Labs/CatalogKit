# Code Of Conduct

This project aims to stay useful, direct, and respectful.

## Expected Behavior

- Be respectful and professional.
- Assume good intent, but be clear and specific.
- Keep technical disagreement focused on the code, contract, and user impact.
- Accept feedback and course-correct quickly when shown evidence.

## Unacceptable Behavior

- Harassment, abuse, or personal attacks
- Discriminatory language or behavior
- Deliberate intimidation or disruption
- Publishing private information without permission

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors
whose behavior makes the project unsafe or unproductive.

## Reporting

If you encounter a serious conduct issue, contact the project maintainers
through the repository hosting platform instead of escalating in public threads.
